## 🔴 Critical (치명적) - 즉시 수정 필요

### 1. **데이터 패널이 항상 보이지 않음 (데스크톱)**
**파일**: `story_chat_screen_v2.dart`
**문제**:
- 데스크톱에서 데이터 패널이 고정되어 있지만, 채팅하다가 잊어버릴 수 있음
- 데이터를 보면서 채팅해야 하는데, 왼쪽 패널에 주목하지 않을 가능성

**제안**:
- 데이터 패널을 접었다 펼 수 있게 (Collapsible)
- 또는 최소화 버튼 추가
- 중요 데이터가 업데이트되면 알림 표시

**우선순위**: Medium → High (사용성에 영향)

---

### 2. **모바일에서 Drawer를 닫고 나면 데이터를 다시 보기 어려움**
**파일**: `story_chat_screen_v2.dart`
**문제**:
- Drawer를 열어서 데이터를 봐야 하는데, 매번 햄버거 메뉴를 눌러야 함
- 채팅 중간에 "Shadow 승률이 뭐였지?" 하고 확인하려면 번거로움

**제안**:
- 떠 있는 FAB (Floating Action Button)으로 "데이터 보기" 추가
- 또는 하단에 미니 데이터 요약 카드 (펼치면 전체 보기)
- 스와이프로 빠르게 Drawer 열기 (이미 있을 수도 있음)

**우선순위**: High

## 🟡 High (높음) - 사용성 개선 필요

**파일**: `story_chat_screen_v2.dart` (line 610-615)
**문제**:

**제안**:
- 항상 하단에 접을 수 있는 "추천 질문" 섹션 유지
- 또는 입력창 위에 작은 버튼 ("💡 도움말")
