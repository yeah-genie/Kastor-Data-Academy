#### ✅ 1. 로딩 인디케이터 추가
**위치**: `main.dart`, `episode1_demo_screen.dart`

**구현 내용**:
- 앱 초기 로딩 시 스피너와 "Loading..." 텍스트 표시
- 차트 로딩 중 CircularProgressIndicator 표시
- 버튼 클릭 시 로딩 상태 시각화
- "차트를 불러오는 중..." 메시지 추가

**효과**:
- 사용자에게 대기 중임을 명확히 전달
- 앱이 응답 없이 멈춘 것처럼 보이는 현상 방지
- 사용자 경험 개선

#### ✅ 2. 버튼 중복 클릭 방지
**위치**: `main.dart`, `episode1_demo_screen.dart`, `story_chat_screen_v2.dart`

**구현 내용**:
- `_isNavigating` 상태 변수로 네비게이션 중복 방지
- 디바운싱 적용 (300ms)
- 선택지 버튼 클릭 후 500ms 동안 비활성화
- 클릭 시 로딩 인디케이터 표시

**효과**:
- 중복 네비게이션 방지
- 의도하지 않은 화면 전환 차단
- 안정적인 사용자 경험

**효과**:
- 첫 방문자가 앱의 목적을 즉시 이해
- 무엇을 해야 하는지 명확한 안내
- 이탈률 감소
