## 📉 이탈 패턴: The Dropout Funnel

### 단계별 이탈 현황

```
계정 생성       10,000명 (100.0%) ──┐
                                    │ 40% 이탈
Titanic 시작     6,000명 ( 60.0%) ──┤
                                    │ 75% 이탈 (제출 안함!)
첫 제출          1,500명 ( 15.0%) ──┤
                                    │ 47% 이탈
5회 이상 제출      800명 (  8.0%) ──┤
                                    │ 63% 이탈
두 번째 Competition 300명 (  3.0%) ──┤
                                    │ 50% 이탈
정기 참여자        150명 (  1.5%) ──┘
```

### 🔴 크리티컬 드롭 포인트

1. **Titanic 시작 → 첫 제출: 85% 이탈**
   - 10명 중 8.5명이 시작하고도 한 번도 제출 안함
   - **민준 페르소나가 경험할 확률: 85%**

2. **첫 Competition만 하고 끝: 97% 이탈**
   - Titanic 완료 후 다른 프로젝트로 넘어가는 비율: 3%
   - **"이거 하나 해봤는데 너무 어려워서 관뒀어요"**

## 💡 민준 페르소나 인사이트

### 민준의 프로필
- **19세, 대학교 1학년, 경영학과 (비전공자)**
- **목표:** 데이터 사이언티스트
- **현재:** Python에서 좌절
- **예산:** 월 50만원 용돈 (가처분 소득 10-15만원)
- **포기 경험:** YouTube 3강 포기, Coursera 1주차 포기

### 🎯 민준이 Kaggle Titanic에서 겪을 문제

#### 1. Python 기초 부족 (75% 확률로 겪음)
```python
# 민준이 막히는 부분
train_data['Age'].fillna(train_data['Age'].median(), inplace=True)
# "fillna가 뭐지? median은? inplace는 왜 True야?"
```
- **소요 시간:** 20시간 추가 학습 필요
- **민준의 현실:** "20시간? 이번 주에 중간고사인데..."
- **결과:** **30% 확률로 여기서 포기**

#### 2. Feature Engineering (80% 확률로 겪음)
```python
# 민준이 이해 못하는 부분
train_data['FamilySize'] = train_data['SibSp'] + train_data['Parch'] + 1
train_data['IsAlone'] = (train_data['FamilySize'] == 1).astype(int)
# "왜 이렇게 해야 돼? 강의에서 안 배웠는데?"
```
- **소요 시간:** 40시간 추가 학습 필요
- **민준의 현실:** "이건 Coursera에서도 안 나왔잖아!"
- **결과:** **35% 확률로 여기서 포기**

#### 3. 예산 문제: 유료 강의 vs 무료 자료
- **Udemy Python 강의:** ₩55,000 (세일 시)
- **Coursera 데이터 사이언스:** ₩49,000/월
- **민준의 예산:** ₩100,000~150,000/월
- **문제:** 1-2개 강의만 결제 가능, 여러 분야 학습 불가능

## 🎓 성공하는 1.5%의 특징

### 정기 참여자로 성장한 사람들의 공통점

1. **체계적인 학습 경로** (60%)
   - 강의 → 미니프로젝트 → Kaggle 순서 준수

4. **적절한 리소스** (35%)
   - 유료 강의 + 무료 자료 조합
   - 경제적 여유
